Connect to Google Sheets

This app pulls schedule and teacher data live from a Google Sheet you control. One-time setup takes about 2 minutes.

1
Create a Google Sheet with two tabs: one for your schedule export, one for teacher profiles (columns: Name, Branches, AgeMin, Notes).
2
Publish each tab separately: File → Share → Publish to web → select each tab individually → CSV → Publish. You'll get two separate URLs.
3
Paste both URLs below. Teachers URL is optional — leave blank if you don't have one yet.
